# elegoo-satellite

[![Chocolatey Version](https://img.shields.io/chocolatey/v/elegoo-satellite.svg)](https://community.chocolatey.org/packages/elegoo-satellite)

**ELEGOO SatelLite** is the official 3D slicing and printing management software from Elegoo.

## Installation

```powershell
choco install elegoo-satellite -y
```
## Upgrade

```PowerShell
choco upgrade elegoo-satellite -y
```
