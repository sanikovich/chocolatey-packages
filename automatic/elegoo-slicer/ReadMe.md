# elegoo-slicer

[![Chocolatey Version](https://img.shields.io/chocolatey/v/elegoo-slicer.svg)](https://community.chocolatey.org/packages/elegoo-slicer)

**Elegoo Slicer** is a powerful 3D slicing software based on PrusaSlicer, optimized for Elegoo 3D printers.

## Installation

```powershell
choco install elegoo-slicer -y
```

## Upgrade

```PowerShell
choco upgrade elegoo-slicer -y
```
