# anycubic-slicer-next

[![Chocolatey Version](https://img.shields.io/chocolatey/v/anycubic-slicer-next.svg)](https://community.chocolatey.org/packages/anycubic-slicer-next)

**Anycubic Slicer Next** (Orca Version) is a powerful 3D slicing software developed by Anycubic, based on Orca Slicer with custom profiles for Anycubic printers.

## Installation

```powershell
choco install anycubic-slicer-next -y
```

## Upgrade



```PowerShell
choco upgrade anycubic-slicer-next -y
```
