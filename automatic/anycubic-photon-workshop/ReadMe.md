# anycubic-photon-workshop

[![Chocolatey Version](https://img.shields.io/chocolatey/v/anycubic-photon-workshop.svg)](https://community.chocolatey.org/packages/anycubic-photon-workshop)

**Anycubic Photon Workshop** is the official slicing software for Anycubic Photon series resin 3D printers.

## Installation

```powershell
choco install anycubic-photon-workshop -y
```
## Upgrade

```PowerShell
choco upgrade anycubic-photon-workshop -y
```
