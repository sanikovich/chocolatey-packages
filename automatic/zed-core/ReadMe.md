# zed-core

[![Chocolatey Version](https://img.shields.io/chocolatey/v/zed-core.svg)](https://community.chocolatey.org/packages/zed-core)
[![Chocolatey Downloads](https://img.shields.io/chocolatey/dt/zed-core.svg)](https://community.chocolatey.org/packages/zed-core)

**Zed** is a high-performance, GPU-accelerated code editor built in Rust. This package installs the official Windows version of Zed.

## Installation

```powershell
choco install zed-core -y
Upgrade
PowerShellchoco upgrade zed-core -y
```

## Features

Uses the official Zed-x86_64.exe installer from GitHub releases
Silent installation (/VERYSILENT)
Automatically updated via Chocolatey-AU

## Automatic Updates
This package is configured for automatic updates using the Chocolatey Automatic Updater (AU).
It checks for new releases on Zed's GitHub repository and updates version, URL, and checksum automatically.
