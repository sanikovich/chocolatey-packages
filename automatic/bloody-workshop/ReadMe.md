# bloody-workshop

[![Chocolatey Version](https://img.shields.io/chocolatey/v/bloody-workshop.svg)](https://community.chocolatey.org/packages/bloody-workshop)

**Bloody WorkShop** is the official software for configuring Bloody gaming mice and keyboards (A4Tech).

## Installation

```powershell
choco install bloody-workshop -y
```
## Upgrade

```powershell
choco upgrade bloody-workshop -y
```
